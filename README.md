# 📝 Todo Web App

A simple and user-friendly todo app to manage daily tasks using HTML, CSS, and JavaScript.

## 🚀 Features

* Add and delete tasks
* Mark tasks as completed
* Clean and simple UI

## 🛠️ Tech

HTML, CSS, JavaScript

## ▶️ Run

Open `index.html` in your browser.
